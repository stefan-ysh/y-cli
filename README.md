# 制作自己的脚手架

## 相关依赖

1. `commander`
   可以自定义一些命令行指令，在输入自定义的命令行的时候，会去执行相应的操作，如：

2. `inquirer`
   可以在命令行询问用户问题，并且可以记录用户回答选择的结果

3. `fs-extra`
   是 fs 的一个扩展，提供了非常多的便利 API，并且继承了 fs 所有方法和为 fs 方法添加了 promise 的支持。

4. `chalk`
   可以美化终端的输出

5. `figlet`
   可以在终端输出 logo

6. `ora`
   控制台的 loading 样式

7. `download-git-repo`
   下载远程模板，并且将模板解压到当前目录

## 目前支持模板

1. Vue2
2. Vue3
3. React
4. React-TS
